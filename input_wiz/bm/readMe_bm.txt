Read me for bm data files

File name
grassland_harvestDate_bm
Ex: g4_20180510_bm

field_id = Grassland field id

fp_id = Field subplot id (This changes with harvest number)
			1XX for G1 (101 - 120)
			2XX for G2 (201 - 220)
			3XX for G3 (301 - 320, 321 - 340)
			4XX for G4 (401 - 420, 421 - 440, 441 - 460)
			
lab_no = Lab sampling number for the data (This is constant for whole year)

harvest = Harvest number

fb = Total fresh biomass from the 1 sq.m subplot (Unit is kg per sq.m)

fb_sample = Sample fresh biomass weight to analyses dry biomass (Unit is kg per sq.m)

db_sample = Sample dry biomass weight after 48 hours in the 105 C oven (Unit is kg per sq.m)

db_p = Dry matter percentage calculated using fb_sample and db sample

db = Total dry biomass from the 1 sq.m subplot (Unit is kg per sq.m)
